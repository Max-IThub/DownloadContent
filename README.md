# DownloadContent

||ВАЖНО||

1)Перед эксплуатацией убедитесь что у вас установлен менеджер пакетов NPM (Инструкция -> https://www.npmjs.com/package/npm);
2)Перед эксплуатацией убедитесь что у вас установлен Node.js издания LTS (Скачать -> https://nodejs.org/ru);
3)Так же перед эксплуатацией перейдите в корень проекта и вставьте в терминал команду (npm install express);
4)По желанию можете установить Live Server для удобства в дальнейшем (Инструкция -> https://visualstudiocode1.ru/ustanovka-i-nastrojka-liveserver-v-vs-code.html);
5)Если у вас не работает приложение -> отключите брандмаузер

||ИНСТРУКЦИЯ ПО ЭКСПЛУАТАЦИИ||

1)Скачайте этот репозиторий;
2)Распакуйте его в любое удобное место;
3)Перейдите в текстовый редактор (прим. Visual Studio Code);
4)Выполните все пункты из блока |ВАЖНО| который описан выше;
5)Запустите файл server.js;
6)После запуска сервера в поле OUTPUT снизу будет указано "Сервер запущен по адресу: http://127.0.0.1:5501";
7)Зажмите ctrl и нажмите левой кнопкой мыши на ссылку "http://127.0.0.1:5501";
8)Если все выполнено верно то вы перейдёте на страницу веб-приложения;
9)В поле ввода введите один из трёх ключевых слов "first, second или third";
10)Нажмите кнопку "Search";
11)Выберите интересующую ссылку в списке;
12)Нажмите кнопку на клавиатуре "f12" и посмотрите в консоль разработчика -> там будет находиться скачанный контент;
